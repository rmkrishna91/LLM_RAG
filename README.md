# LLM with RAG

A Retrieval-Augmented Generation (RAG) pipeline using:

- 🧠 **Cohere Command R** for language generation  
- 🔍 **Cohere Embedding v3.0** for semantic search  
- 🧱 **Chroma** as the vector store  
- ⚙️ **LangChain** to orchestrate everything
